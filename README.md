# Super Bowl TiTi-apps (GitHub Pages)

## Publicar en GitHub Pages
1. Sube estos archivos al repo (rama `main`).
2. Ve a **Settings** → **Pages**
3. En **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
4. Guarda.

En 1–2 minutos tu sitio quedará disponible en la URL que te muestre GitHub Pages.
